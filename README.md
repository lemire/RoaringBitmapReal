RoaringBitmapReal
=================

Code for real-data experiments on roaring bitmaps as prepared by O. Kaser
for the paper:

Samy Chambi, Daniel Lemire, Owen Kaser, Robert Godin,
Better bitmap performance with Roaring bitmaps,
http://arxiv.org/abs/1402.6407


Usage 
-------

* install java
* install maven 2
